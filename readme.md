# Gestionnaire de Tâches - BTS SIO

## Description
Application web permettant de créer, gérer et supprimer des tâches. 
Ce projet a été réalisé dans le cadre de l'évaluation HTML/CSS/JS.

## Fonctionnalités
- Ajout de tâches via un formulaire.
- Stockage dynamique dans un tableau JavaScript.
- Marquage d'une tâche comme "complétée" (visuel distinct).
- Suppression d'une tâche du DOM et du tableau.

## Technologies utilisées
- **HTML5** : Structure de la page et du formulaire.
- **CSS3** : Mise en page.
- **JavaScript** : Manipulation du DOM et gestion des événements.
